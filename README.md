###  California Housing Prices Prediction
*A regression project to predict house prices in California based on the 1990 census data.*

#### **Workflow & Steps:**
* **Data Exploration:** Analyzed dataset features such as median income, house age, and average rooms.
* **Data Visualization:** Created a **Histogram** to understand feature distributions and identify patterns.
* **Correlation Analysis:** Used a **Heatmap** to find relationships between features and the target variable (Median House Value).
* **Data Splitting:** Divided data into Training (80%) and Testing (20%) sets for robust evaluation.
* **Feature Scaling:** Applied `StandardScaler` to normalize features for improved model stability.
* **Model:** Developed and trained a **Random Forest Regressor**.
* **Performance:** Achieved an **82% R-squared score** on the test set.
